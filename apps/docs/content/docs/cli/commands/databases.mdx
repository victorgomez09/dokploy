---
title: Databases
description: A guide to using the Alvi Paas CLI to manage databases
---

The Alvi Paas CLI can be used to create, deploy, and manage databases.

## Requirements

Is required to be already authenticated with the Alvi Paas CLI.

## Commands

### MariaDB

1. `alvi-paas database mariadb create` - Create a new mariadb database.
2. `alvi-paas database mariadb delete` - Delete an mariadb database.
3. `alvi-paas database mariadb deploy` - Deploy a mariadb database.
4. `alvi-paas database mariadb stop` - Stop a running mariadb database.
