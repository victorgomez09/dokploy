---

The Alvi Paas CLI can be used to create, deploy, and manage projects.

## Requirements

Is required to be already authenticated with the Alvi Paas CLI.

## Commands

1. `alvi-paas project create` - Create a new project.
2. `alvi-paas project info` - Get information about a project.
3. `alvi-paas project list` - List all projects.
