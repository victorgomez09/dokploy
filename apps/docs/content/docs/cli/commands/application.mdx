The Alvi Paas CLI can be used to create, deploy, and manage applications.

## Requirements

Is required to be already authenticated with the Alvi Paas CLI.

## Commands

1. `alvi-paas app create` - Create a new application.
2. `alvi-paas app delete` - Delete an application.
3. `alvi-paas app deploy` - Deploy an application.
4. `alvi-paas app stop` - Stop a running application.
