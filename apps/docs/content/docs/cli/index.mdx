---

Alvi Paas CLI is a command-line tool for remotely managing your Alvi Paas server. It simplifies creating, deploying, and managing applications and databases.

## Installation

```bash
npm install -g @alvi-paas/cli
```

## Usage

```bash
alvi-paas COMMAND
```

To get help on a specific command:

```bash
alvi-paas COMMAND --help
```
